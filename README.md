NO.1  三星笔记注意事项     Some issue about SamsungNotes:

1. 不能登录账号或同步   

2. 不能直接插入图片和附件，但可以把相册开成小窗来长按拖入   

3. 对于需要导入的.sdoc格式文件，可在文件管理中使用“分享”或“选择其他应用打开”来导入    

4. 编辑文稿之前先测试一下能否正常导出文件，以免文稿白写

5. 声明：三星笔记安装包并非由我修改，是某位匿名大佬的杰作，目前我也不知道是谁。如果涉及版权，请在 issue 里留言告诉我。     

1. Can not login and sync notes with samsung account.

2. Can not call photos app to select picture, but can manually start photo app as a float window, then drag your target picture & drop in SamsungNotes.

3. If need import a .sdoc file, please use "share" or "open in other app" in a file manage app.

4. Test if it can successfully export file before wirtting, avoid letting your efforts go waste.

NO.2  OneNote注意事项   Some issue about OneNote: 

1. 这是一个旧版本，可以使用便笺功能。初次打开时，仍会提示无法使用，但这个软件放在手机上一段时间不管他，它会自己云控更新成新版，而这个新版是可以使用便笺的。注意：并非所有系统都能触发更新，详见下一条说明。

2. 可触发更新的设备或系统：1.华为设备(鸿蒙2.0-4.3 全部过测) 2.小米设备(安卓15澎湃os2 过测)， 备注：华为和小米更新出的新界面甚至是不一样的

3. 无法触发更新的设备或系统：1.三星设备(OneUI4.1 - OneUI7 已确认失败)

1. It's an old version which can use Sticky Notes. When your first time launch the app, it still indicates unavailable. But if you leave it on your phone, it will automatically update to a new version which can use Sticky Notes, may be several days.(By it self, Not by any other app store, DISABLE your auto-update!). Not all system can trigger this update, check next for detail.

2. Devices or systems CAN trigger upadate: 1.Huawei(HarmonyOS2.0 - 4.3 by Android10-12 Passed) 2.Xiaomi(HyperOS2.0 by Android 15 Passed). They even have different new page.

3. Devices or systems CANNOT trigger upadate: 1.Samsung(OneUI4.1 - OneuUI7 Failed)
